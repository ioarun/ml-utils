# ml-utils
Helpful ML functions.

## Dependencies
* Python 3.6.9
* Pip3

## Installation
```
pip install .
```

## Usage
```
from mlutils import check_onnx_model

PYTORCH_MODEL_PATH = path/to/model.pth

check_onnx_model(PYTORCH_MODEL_PATH)

```
